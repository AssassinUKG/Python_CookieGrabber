# Python 3 HTTP Cookie Grabber
A cookie grabber for webrequests

## Usage

A simple cookie grabber with some formatting.

Start up
```
python3 cookieGrabber.py
```

Set port
```
python3 cookieGrabber.py -p 9999
```
